BOM Component Description Enricher

Application Documentation

Description

The BOM Component Description Enricher is designed to simplify and automate work with Excel files. It enables quick data enrichment, standardizes component descriptions, and manages processing results. AI integration ensures accuracy and relevance of information, making data processing efficient and reliable.

Main Features

  • Excel File Loading and Analysis: Easily upload your Excel files and get content information
  • Data Enrichment: Automatic search for additional component information using artificial intelligence
  • Description Standardization: Transform and unify component descriptions according to set rules
  • Results Management: Store and access intermediate and final results
  • Intuitive Interface: Simple and clear management without programming requirements

How to Start

Step 1: Registration and Login

  • Registration: Go to the registration page and create an account by providing the required information
  • Login: After registration, log in to the system using your credentials

Step 2: Upload Excel File

  • Go to Upload Section: On the main page, find and click the "Upload File" button
  • Select File: A file selection window will open. Find and select the desired Excel file on your computer
  • Start Upload: After selecting the file, the upload process will begin. Wait for the upload to complete

Step 3: Configure Processing Parameters

  • Sheet Selection: If your Excel file contains multiple sheets, select the one you want to process
  • Column Specification:
    • Part Number Column: Specify the column containing part numbers
    • Description Column: Specify the column containing component descriptions
  • File Identifier: Enter a unique identifier for the file so the system can save intermediate results

Step 4: Start Processing

  • Begin Process: Click "Start Processing" to begin the data enrichment and standardization process
  • Track Progress: During processing, you can view the current progress and receive notifications about completed steps
  • Preview: After processing, you can preview changes in component descriptions before saving the final result

Step 5: Download Results

  • Processing Complete: You will receive a notification when the file is ready
  • Download Enriched File: Click the "Download" button to save the enriched Excel file to your device
  • Access Intermediate Results: If needed, you can download intermediate versions of the file from the "My Files" section

File Management

Storage and Access

  • Intermediate Results: The system automatically saves intermediate results every 10 processed rows, allowing process recovery in case of failures
  • Storage Limits: Storage is limited to the last 10 versions of files
  • View Available Files: In the "Processed files" section, you can see all available versions of your processed files

Frequently Asked Questions

What Excel formats are supported?

Our application supports standard Excel file formats, including .xlsx and .xls.

What volume of data can be processed?

The application efficiently processes files with several thousand rows. However, for optimal performance, we recommend splitting very large files into smaller parts.

How is data security ensured?

We use modern encryption and storage methods to ensure the security and confidentiality of your files. Your data is not shared with third parties.

What if I find an error in a component description?

If you notice an error, you can reprocess the file or manually edit the descriptions in the downloaded Excel file.
